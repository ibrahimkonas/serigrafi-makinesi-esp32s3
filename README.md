# serigrafi_mapp_clean

Bu proje, ibrahimkonas/serigrafi-makinesi-esp32s3 GitHub reposundaki serigrafi_app/serigrafi_mapp klasörünün güncel ve temiz Flutter kodlarını içerir.

## Kurulum

1. Flutter SDK kurulu olmalı.
2. Gerekli bağımlılıkları yüklemek için proje kök dizininde:

   flutter pub get

3. Uygulamayı başlatmak için:

   flutter run

## İçerik
- lib/main.dart
- lib/src/widgets/
- lib/src/models/
- assets/
- web/

## Notlar
- Kodlar doğrudan orijinal GitHub deposundan alınmıştır.
- Gerekli ek ayar ve bağımlılıklar için pubspec.yaml dosyasını ekleyin.
